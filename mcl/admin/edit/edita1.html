EDIT : 메인 배너

진행중인 배너

선택된 배너

권장 사이즈 : 960*960
  1. 메인화면 상단에 나오는 배너입니다.
  2. 이미지를 드래그하여 순서를 변경할 수 있습니다.
  3. 변경을 원하는 이미지를 선택하면 선택된 배너가 나옵니다.
  4. 업로드할 이미지 파일을 선택합니다.
  5. epi://에피소드번호를 입력합니다. 예)epi://16
  6. 적용하기 버튼을 누르면 적용됩니다.
